# Issuer
본인인증이 완료된 Holder에게 `VC(Verifiable Credential)`를 발급 해줍니다.
<br/>
<br/>
## VC(Verifiable Credential) 발급 과정
### step1 - 본인 인증
- `Holder`가 Issuer에게 본인 인증 요청
- `Issuer`는 Holder의 신원 정보를 조회
<br/>

### step2 - VC 발급
- 본인인증에 성공한 Holder에게 `VC를 발급`
- Holder가 전달한 publicKey를 블록체인 서버에 전달
